# my-first-repo

## Overview
A minimal personal "Hello World" project imported from GitHub. It consists of a
single HTML page (in Russian) showing "Привет, мир!" and a couple of paragraphs.
There is no backend, framework, or build step.

## Structure
- `index.html` — the page served to visitors (created during Replit setup so the
  site has a proper entry point; wraps the original content in a full HTML document).
- `<h1>Привет, мир!</h1>.html` — the original file from the imported repo, kept
  as-is for reference/history. Not served directly.

## Running the project
A workflow ("Start application") runs `python3 -m http.server 5000` to serve the
static `index.html` file. This is the simplest way to preview a plain HTML page
on Replit; no dependencies or build steps are required.
